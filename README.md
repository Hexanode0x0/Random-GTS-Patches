# General
A repo for random patches for GTS or GTS CE.

Might contain work in progress stuff or experimental things.

# Race Patches
All race patches here require Community Edition R82 or higher.

COTR was made on the UUNP version, but I don't think the plugin is any different between it and CBBE. It requires RaceCompatibility.esm from the RCS patch hub. Do not pick any other patch.

LORN requires the Aetherius and Scion patches in the FOMOD. It's an AIO patch for all races, so select all of them in the FOMOD.
